# Code of Conduct | 行为准则

## Our Standard | 我们的基本标准

We want this repository to be a respectful, practical, and collaborative place for sharing reusable project systems.  
我们希望这个仓库是一个尊重他人、务实协作、适合分享可复用项目系统的地方。

Examples of behavior that helps:  
以下行为有助于营造良好的协作环境：

- being clear and constructive  
  表达清楚，给出建设性反馈
- assuming good intent while giving direct feedback  
  在直接反馈的同时，默认对方是出于善意
- documenting decisions so others can build on them  
  把决策记录清楚，方便其他人继续往下推进
- keeping discussion focused on improving the toolkit  
  让讨论聚焦在如何改进工具包本身

Examples of unacceptable behavior:  
以下行为不可接受：

- harassment or personal attacks  
  骚扰、攻击或针对个人的言行
- dismissive or hostile communication  
  轻蔑、敌对或带攻击性的沟通方式
- intentionally misleading documentation or release information  
  故意提供误导性的文档或发布信息
- sharing private or sensitive project information without permission  
  未经允许公开私密或敏感项目信息

## Enforcement | 执行方式

Project maintainers may remove, edit, or reject contributions and interactions that do not support a safe and productive collaboration environment.  
对于不利于安全、有效协作环境的贡献或互动，项目维护者可以进行编辑、移除或拒绝处理。

## Scope | 适用范围

This code of conduct applies to repository discussions, issues, pull requests, and other project spaces tied to this repository.  
本行为准则适用于仓库讨论区、issues、pull requests，以及其他与本仓库相关的协作空间。
