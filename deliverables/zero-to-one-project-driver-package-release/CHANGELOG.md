# Changelog

## v1.0.0 - 2026-03-17

### Added
- `zero-to-one-project-driver` main orchestration skill
- role skills for product, research, delivery, QA, archive, product ops, UI, and GTM
- role skill framework and bundle usage guide
- document scaffolding templates for 0-1 projects
- starter repo template with minimal runnable example
- generic Playwright responsive toolkit extracted from the enterprise Poll-Vote test system
- install and scaffold scripts

### Improved
- unified package structure for sharing and onboarding
- clearer separation between project orchestration, starter repo, and validation tooling
- documentation entry points for reading, installation, and usage

### Notes
- This release is intended as a reusable internal toolkit and starter system.
- The bundled Playwright toolkit is configuration-driven and should be adapted per project selectors and base URL.
