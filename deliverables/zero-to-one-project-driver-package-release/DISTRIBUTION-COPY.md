# 发布文案包

## 一句话卖点

一套把 0-1 项目推进方法、角色协作、文档模板、starter repo 和企业级页面验证工具整合在一起的项目操作系统。

## 标准介绍

`Zero to One Project Driver Bundle` 是一套面向 0-1 到 1-10 项目的内部工具包，帮助团队把“项目如何推进、如何分工、如何沉淀、如何验证”系统化。它不只是一个 skill，也不只是一个模板仓库，而是把多角色 skills、项目文档模板、starter repo 骨架和通用 Playwright 响应式测试工具整合成一个可复用、可转交、可扩展的统一工具包。

## 短版介绍

这是一个把项目推进方法、角色 skills、文档模板、starter repo 和 Playwright 验证工具打包在一起的统一工具包，适合新项目快速启动和团队内部复制。

## 仓库首页简介

面向 0-1 到 1-10 项目的统一工具包，包含：

- 多角色项目推进 skills
- 文档模板与项目沉淀框架
- starter repo 最小骨架
- 通用 Playwright 响应式验证工具

适合希望提升项目推进效率、减少上下文丢失、强化并行协作与快速验证的团队。
